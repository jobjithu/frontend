# 🎬 OMDb Movie Search App

## 🔍 Overview
This is a simple web app that lets users search for movies using the [OMDb API](http://www.omdbapi.com/).  
**Endpoints used:**
- `/?s={SEARCH_TERM}&apikey=KEY`: To search for movies.
